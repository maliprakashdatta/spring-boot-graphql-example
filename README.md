## spring-boot-graphql-example 

### Building
 
The example can be built with
```shell
mvn clean install
```

### Running the example in your local
```shell
mvn clean spring-boot:run
```

### Run on Postman graphql
## graphql Link
```shell
http://localhost:8080/graphql
```
